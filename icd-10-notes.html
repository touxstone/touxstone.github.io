Overview.

  • use the tabular index 00:10:13
  • laterality 1: right, 2:left, 3 bilateral
    • If documentation states bilateral but not bilateral code is available code each side separately.
  • Combination: classify 2 diagnoses:
    • A diagnosis with an associated secondary process or
    • An associated complication.
  • Sequela: A residual effect produced after the acute phase of an illness or injury is past.
  • Signs or symptoms codes: Used when the diagnosis has not been established by the end of the encounter
    • Not used for signs and symptoms that are integral to a given diagnosis.
    • If symptoms are present beyond what is integral to a given diagnosis code those symptoms.

Chapter1 A00-B99

  • Certain Infections & Parasitic Diseases.
    • Which appear in so many different diseases so here as entities &|or as appearing so often as combination & multiple coding.
    • B97-B97, Bacterial & Infectious Agents.

    infectious codes in other chapter do not contain causal organism.

    • Codes from this range are used as additional codes to infections coded in other chapters.
    • Infection codes provide instructions referencing B95-B97.
  • Infection resistant to antibiotics.
    • Any documented antibiotic resistance must be coded.
      • Z16, resistance to antimicrobial drugs if infection code does not specify resistance.
    • Z16 codes are not used when methicillin resistance is identify in the MRSA.
    • Do not assign additional B95 category MRSA codes if combination codes are available. Example A41.02 (sepsis due to methicillin resistance staphylococcus aureus)
